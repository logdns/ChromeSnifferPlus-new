# LibSniffer Chrome Web Store Listing

## Short Description

Detect website technologies such as JavaScript frameworks, CDN, security tools, payment processors, font scripts, JavaScript libraries, web frameworks, UI frameworks, and programming languages.

## Detailed Description

LibSniffer helps developers and site operators inspect the technology stack used by the current website. It detects common libraries, frameworks, CMS products, server software, APIs, analytics tools, CDN providers, UI frameworks, font scripts, payment processors, security tools, and programming languages.

The extension runs local detection automatically and supports optional free API lookup providers from the options page. Remote lookup is user-triggered and sends the current page hostname only after you click the API lookup button.

## 中文简介

LibSniffer 可帮助开发者快速识别当前网站使用的技术栈，包括 JavaScript 框架、内容分发网络（CDN）、安全工具、支付处理器、字体脚本、JavaScript 库、Web 框架、用户界面（UI）框架、编程语言、CMS、Web 服务器和分析工具等。

扩展会自动执行本地检测，也支持在选项页配置免费的远程 API 查询。远程查询为手动触发，只有点击 API 查询按钮后才会发送当前页面域名。
